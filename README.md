##Installation
* Build solution
* Run `EmojiSense.exe`.
	* Snippets should be generated in the folder `Snippets`.
* In Visual Studio, open **Tools -> Code Snippets Manager**
* Click **Add...**.
* Browse to the snippets folder and click **Select Folder**.
* Click **OK** .
